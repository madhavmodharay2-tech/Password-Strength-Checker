# Password-Strength-Checker

How to use direction

A simple web-based application to evaluate password strength.

1. Download or clone this repository to your computer.
2. Make sure Python is installed on your system.
3. Open a terminal once and install Flask: `pip install Flask`
4. Double-click the `run.bat` file. The application will automatically launch the backend server and open the web page in your default browser instantly!
